
Задача 1 - Reverse
Ты видишь рабочую программу, в которой массив int[] numbers заполняется числами в методе init().
Затем в методе reverse() происходит перестановка чисел в обратном порядке. Твоя задача — переписать код так,
чтобы вместо массива int[] numbers использовался список ArrayList numbers.
